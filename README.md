# Java × AI 自動化エンジニア

Java 10年超 × 生成AI活用による業務自動化が専門です。  
「人手作業をほぼゼロにするAIワークフロー構築」を得意としています。

## 🛠 技術スタック
**言語**: Java / Python / VBA  
**AI**: Claude API / OpenAI API / プロンプトエンジニアリング  
**フレームワーク**: Spring Boot / Spring Framework  
**ツール**: Claude Code / VS Code / Git  

## 🤖 稼働中のシステム
| システム | 概要 |
|---------|------|
| 副業案件自動スクリーニング | RSS取得 → AIスコアリング → Gmail通知を完全自動化 |
| 株式決算短信 自動Note生成 | 決算短信収集 → AI要約 → 自動投稿（[@YUKI520350](https://x.com/YUKI520350)） |
| 楽天ROOM 投稿文自動生成 | 商品情報取得 → AI訴求文生成 → 投稿データ出力 |
| FX自動売買（自律PDCAエージェント） | PLAN→DO→CHECK→ACTをLLMが自律実行 |

※ 公開リポジトリはデモ版のため一部機能を制限または代替実装しています。  
　 実運用版は個人の副業・投資用途で現在も稼働中です。
